# Orders Feature

Shopping cart, checkout flow, and order history.

## Responsibilities

- Cart management (add, remove, update quantities)
- Checkout with address and payment selection
- Real-time order status tracking
- Order history with re-order capability

## Key Components

- `CartComponent` — sidebar/page cart view
- `CheckoutComponent` — multi-step checkout
- `OrderStatusComponent` — live tracking with timeline
- `OrderHistoryComponent` — past orders list

## Routes

- `/cart` — cart page
- `/checkout` — checkout flow
- `/orders` — order history
- `/orders/:id` — order detail with status
