# Profile Feature

User account management, addresses, and preferences.

## Responsibilities

- View and edit profile information
- Manage saved delivery addresses
- Payment methods management
- Notification preferences

## Key Components

- `ProfileComponent` — account info form
- `AddressListComponent` — CRUD for saved addresses
- `PreferencesComponent` — notification and app settings

## Routes

- `/profile` — profile overview
- `/profile/addresses` — address management
- `/profile/preferences` — settings page
