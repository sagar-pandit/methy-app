# Restaurants Feature

Browse, search, and view restaurant details and menus.

## Responsibilities

- Restaurant listing with filters (cuisine, rating, distance)
- Restaurant detail page with menu categories and items
- Search and sort functionality

## Key Components

- `RestaurantListComponent` — grid/list of restaurants
- `RestaurantDetailComponent` — single restaurant view with menu
- `MenuItemCardComponent` — reusable menu item display

## Routes

- `/restaurants` — listing page
- `/restaurants/:id` — detail page
